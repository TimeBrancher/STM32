This project is a code that i develope while studying the NUCLEO-F767ZI, its make blink the red, blue and green led in that order, repeat the pattern 3 times and finally keep on every led.

This code allow me to practice the project creation with STM32CubeMX, the configuration of MCU using the user manual, and HAL GPIO framework. Also i can experiment with the use of for loop in STM32.

You can clone the repo and check how it work. In case you want to see the code you will find it in main.c file
